Core Curriculum
Financial Foundations™
Ages 11–18 · Builder & Strategist

An 8-unit, standards-aligned curriculum delivered by trained iMPACT Partners™. Instruction responds directly to FLIQ Score™ data — so facilitators know exactly which behavioral patterns to address in each student before the first lesson begins.

Generates
Pre/post FLIQ assessments Facilitator data dashboard Score progression tracking Outcome documentation
Deepens what The $100 Week™ reveals
Student Fintech App
CoinQuest™
Ages 8–14 · Explorer & Builder

A school-safe student banking simulation where learners manage a checking account, set savings goals, and navigate real financial decisions in a private, data-protected environment. No real money. No ads. No personal information required.

Generates
Banking literacy data Savings behavior records Goal-completion rates Spending pattern insights
Extends $100 Week™ concepts into ongoing daily practice
Gamified Assessment
FLIQ Runner™
Ages 8–18 · All Bands

An arcade-style game set in Vela City where students make financial decisions under time pressure. Every in-game choice generates behavioral data that teachers can monitor in real time — making assessment feel like play and eliminating test anxiety entirely.

Generates
Real-time FLIQ Score™ data Decision-speed patterns Classroom discussion prompts Pre/post assessment data
Used as pre-assessment before $100 Week™ or reinforcement after
Character IP & Media
WWK Adventures™
Ages 8–12 · Explorer Band

A book series, read-along portal, and animated content universe featuring The Crew — six kids who navigate real-world financial situations together. Financial concepts are embedded in narrative, building vocabulary and engagement long before formal instruction begins.

Generates
Home engagement Early financial vocabulary Brand recognition Parent-child conversations
The narrative layer that keeps learning alive outside the classroom

iMPACT Partners™
The Practitioner Network
The credential that
unlocks the platform.

iMPACT Partners™ are trained youth workers, educators, and program staff who are certified to deploy WealthWise Kids™ programs. Certification is online, self-paced, and free for qualifying partner organizations. It's how we guarantee program fidelity — and how partners get everything.

What Partners Receive
Cohort FLIQ Score™ Reports — real behavioral data on every student who completes a program
Full Curriculum Access — Financial Foundations™ facilitator guides, session decks, and student materials
iMPACT Certification — credentialed via online module library, stackable across programs
Partner Portal Access — dashboard, session management, reporting, and resource downloads
Dedicated Support — direct WWK contact for onboarding, facilitation questions, and outcomes documentation
Qualifying organizations: Youth-serving nonprofits · Schools & districts · After-school programs · Community centers · Libraries · Faith-based youth programs
